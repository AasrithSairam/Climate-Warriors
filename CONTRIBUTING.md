# Contributing to Climate Warriors

Thank you for your interest in contributing to the Climate Warriors project!

## How to Contribute

1. **Fork the Repository**: Create your own fork of the project.
2. **Clone the Fork**: Clone your fork to your local machine.
3. **Create a Branch**: Create a new branch for your feature or bug fix.
4. **Make Changes**: Implement your changes and ensure they follow the project's coding standards.
5. **Commit Changes**: Commit your changes with clear and descriptive commit messages.
6. **Push Changes**: Push your changes to your fork on GitHub.
7. **Submit a Pull Request**: Submit a pull request to the `main` branch of the original repository.

## Coding Standards

- Follow the established style for each component (Node.js, React, Python).
- Ensure all new features are documented.
- Add tests for any new functionality.

## Team Workflow

We use a standard GitHub flow for reviews and merging.
